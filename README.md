# Daily Quote QR

Минималистичная программа-компаньон, которая каждый день показывает уникальную цитату в виде QR-кода.

## Особенности

- **Ежедневная цитата** — каждый день новая фраза, но в течение дня она остаётся стабильной
- **Персонализация** — у каждого пользователя Windows своя уникальная история цитат
- **Парящая кнопка** — стильная кнопка закрытия с плавной анимацией за пределами окна
- **Высокое качество** — сглаженные элементы интерфейса через PIL
- **Без кэша на диске** — всё вычисляется "на лету"

## Установка

1. Установите Python 3.10+ с [python.org](https://www.python.org/)
2. Установите зависимости:
   ```bash
   pip install pillow qrcode

3. Запустите программу:
    python main.py

## Использование
Перетаскивание — зажмите левую кнопку мыши на QR-коде и двигайте окно
Закрытие — нажмите на круглую кнопку в правом верхнем углу
Смена цитаты — перезапустите программу на следующий день
